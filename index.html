I am a Full-Stack Developer with a passion for collaboration, learning new technologies and using my analytical background to solve problems. Prior to transitioning to development, I spent four and a half years in Financial and Operations Analyst roles for an investment consulting firm. Knowing that it would be applicable to my career, I began taking a Python course in my spare time. The class provided a great introduction to the world of coding and inspired me to pursue software development full time.

I recently completed LEARN Academy’s three month in-class coding bootcamp, where I spent over 480 hours developing full stack web applications using JavaScript, React, Ruby on Rails, PostgreSQL, HTML, and CSS. Following the bootcamp, I interned with Notch8, a development consultancy firm based in San Diego. When I'm not coding, I'm usually exercising, reading or exploring San Diego.
